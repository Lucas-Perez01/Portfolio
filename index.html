Sobre mí

Soy un desarrollador web Frontend con experiencia en la creación de soluciones digitales atractivas y funcionales. He trabajado en proyectos como una página web e-commerce y una aplicación del clima, donde apliqué mis conocimientos para construir interfaces de usuario intuitivas.
